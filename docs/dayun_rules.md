# 大运顺逆排规则 — 完整文档

> **最后更新**：2026-05-15
> **版本**：v4.2 — 支持日干派/年干派双模式

---

## 一、两种门派

### 1.1 日干派（默认，主流）
- **依据**：《渊海子平》《三命通会》等子平派主流
- **规则**：用**日柱天干**的阴阳 + 性别决定顺逆
- **市场占比**：绝大多数排盘网站和命理师

### 1.2 年干派（少数）
- **依据**：部分民间门派
- **规则**：用**年柱天干**的阴阳 + 性别决定顺逆
- **市场占比**：少数排盘网站（如 suanzhun.net）

### 1.3 系统行为
- **默认**：日干派（`dayun_mode="day_gan"`）
- **切换**：输入参数 `dayun_mode="year_gan"` 切换为年干派
- **输出标注**：报告中明确标注使用了哪种排法

## 二、核心规则

### 2.1 日干派（默认）：由【日干阴阳 + 性别】决定

| 日干阴阳 | 性别 | 大运方向 | 简称 |
|----------|------|----------|------|
| 阳干（甲丙戊庚壬） | 男 | 顺排 | 阳男顺 |
| 阴干（乙丁己辛癸） | 女 | 顺排 | 阴女顺 |
| 阴干（乙丁己辛癸） | 男 | 逆排 | 阴男逆 |
| 阳干（甲丙戊庚壬） | 女 | 逆排 | 阳女逆 |

**口诀**：阳男阴女顺行，阴男阳女逆行

**判断顺序**：
1. 确定日柱天干（万年历查表）
2. 判断日干是阳还是阴（甲丙戊庚壬=阳，乙丁己辛癸=阴）
3. 结合性别查上表

### 2.2 年干派：由【年干阴阳 + 性别】决定

| 年干阴阳 | 性别 | 大运方向 | 简称 |
|----------|------|----------|------|
| 阳干（甲丙戊庚壬） | 男 | 顺排 | 阳男顺 |
| 阴干（乙丁己辛癸） | 女 | 顺排 | 阴女顺 |
| 阴干（乙丁己辛癸） | 男 | 逆排 | 阴男逆 |
| 阳干（甲丙戊庚壬） | 女 | 逆排 | 阳女逆 |

> 注意：规则形式相同，只是判断基准从"日干"变为"年干"。

### 2.3 起运年龄计算（两种派别共用）

#### 顺排（阳干+同性别 / 阴干+异性）
- 从出生日**向后**数到下一个节气的天数
- 起运年龄 = 天数 ÷ 3（向下取整）
- 三天折一年

#### 逆排（阴干+同性别 / 阳干+异性）
- 从出生日**向前**数到上一个节气的天数
- 起运年龄 = 天数 ÷ 3（向下取整）
- 三天折一年

**关键**：用的是**节气**（24节气），不是中气。每阳历月取第一个节气。

### 2.4 大运干支序列（两种派别共用）

大运干支从**月柱**出发，按方向递推，每10年一步：

#### 顺排
- 天干：(月干序号 + 1 + i) % 10
- 地支：(月支序号 + 1 + i) % 12

#### 逆排
- 天干：(月干序号 - 1 - i) % 10
- 地支：(月支序号 - 1 - i) % 12

其中 i = 0, 1, 2, ... (步数索引，从0开始)

## 三、Golden Test Case 验证

### 输入
- 出生日期：1974年7月5日
- 性别：男
- 时辰：己酉时（17:00）
- 四柱：甲寅 庚午 丁未 己酉
- 日干：**丁**（阴干）
- 年干：**甲**（阳干）
- 月柱：庚午

### 3.1 日干派推导（默认）

**Step 1 — 大运方向**
- 日干丁 = 阴干
- 性别 = 男
- 阴男 → **逆排** ✅

**Step 2 — 起运年龄**
- 逆排，找上一个节气
- 1974/7/5 上一个节气 = 芒种(6/5)
- 距离天数 = 7月5日 - 6月5日 = 30天
- 起运年龄 = 30 ÷ 3 = **10岁**
- 起运年份 = 1974 + 10 = **1984年**

**Step 3 — 大运序列（逆排，从庚午出发）**

| 步数 | 干支 | 年龄范围 | 公历年份 |
|------|------|----------|----------|
| 1 | 己巳 | 10-19岁 | 1984-1993 |
| 2 | 戊辰 | 20-29岁 | 1994-2003 |
| 3 | 丁卯 | 30-39岁 | 2004-2013 |
| 4 | 丙寅 | 40-49岁 | 2014-2023 |
| **5** | **乙丑** | **50-59岁** | **2024-2033** ← 当前 |
| 6 | 甲子 | 60-69岁 | 2034-2043 |

**日干派结论：2026年 = 乙丑大运** ✅

### 3.2 年干派推导

**Step 1 — 大运方向**
- 年干甲 = 阳干
- 性别 = 男
- 阳男 → **顺排** ✅

**Step 2 — 起运年龄**
- 顺排，找下一个节气
- 1974/7/5 下一个节气 = 小暑(7/7)
- 距离天数 = 7月7日 - 7月5日 = 2天
- 起运年龄 = 2 ÷ 3 = **0岁**
- 起运年份 = 1974 + 0 = **1974年**

**Step 3 — 大运序列（顺排，从庚午出发）**

| 步数 | 干支 | 年龄范围 | 公历年份 |
|------|------|----------|----------|
| 1 | 辛未 | 0-9岁 | 1974-1983 |
| 2 | 壬申 | 10-19岁 | 1984-1993 |
| 3 | 癸酉 | 20-29岁 | 1994-2003 |
| 4 | 甲戌 | 30-39岁 | 2004-2013 |
| 5 | 乙亥 | 40-49岁 | 2014-2023 |
| **6** | **丙子** | **50-59岁** | **2024-2033** ← 当前 |
| 7 | 丁丑 | 60-69岁 | 2034-2043 |

**年干派结论：2026年 = 丙子大运** ✅

## 六、常见错误防范

1. **❌ 混淆日干派和年干派** → 明确标注使用了哪种
2. **❌ 手数天干地支步数** → 必须用公式计算或查表
3. **❌ 用中气判断起运** → 必须用节气（12个）
4. **❌ 忽略同月两个节气** → 每月取第一个（如6月取芒种非夏至）
5. **❌ 混淆周岁和虚岁** → 代码统一使用周岁（公历年份差）
6. **❌ 大运序列不从月柱出发** → 必须从月柱出发顺/逆推
